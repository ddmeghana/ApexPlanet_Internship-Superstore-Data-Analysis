| Column Name   | Description                                                                |
| ------------- | -------------------------------------------------------------------------- |
| Row_ID        | Unique identifier for each row/record                                      |
| Order_ID      | Unique ID assigned to each order                                           |
| Order_Date    | Date when the order was placed                                             |
| Ship_Date     | Date when the order was shipped                                            |
| Ship_Mode     | Shipping method used (First Class, Second Class, Standard Class, Same Day) |
| Customer_ID   | Unique identifier for each customer                                        |
| Customer_Name | Full name of the customer                                                  |
| Segment       | Customer segment (Consumer, Corporate, Home Office)                        |
| Country       | Country where the order was placed                                         |
| City          | City where the order was placed                                            |
| State         | State where the order was placed                                           |
| Postal_Code   | Postal/ZIP code of the location                                            |
| Region        | Geographic region (East, West, Central, South)                             |
| Product_ID    | Unique identifier for each product                                         |
| Category      | High-level product category (Furniture, Office Supplies, Technology)       |
| Sub_Category  | Sub-division of category (e.g., Chairs, Phones, Binders)                   |
| Product_Name  | Name/description of the product                                            |
| Sales         | Total revenue generated from the order                                     |
| Quantity      | Number of units sold                                                       |
| Discount      | Discount applied on the product (in percentage/decimal form)               |
| Profit        | Profit earned from the order                                               |
